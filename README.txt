To-Do List Application using Python

Description:
This project is a To-Do List application developed using Python. It provides both Command-Line Interface (CLI) and Graphical User Interface (GUI) to manage tasks efficiently.

Features:

* Add tasks
* View tasks
* Update tasks
* Delete tasks
* GUI-based interface using Tkinter
* Persistent storage using file handling

Technologies Used:

* Python
* Tkinter (for GUI)
* File Handling

How to Run:

1. Open terminal in project folder

2. Run GUI version:
   python todo_gui.py

3. (Optional) Run CLI version:
   python todo.py

Outcome:
This project helps users organize tasks effectively and demonstrates Python programming, GUI development, and file handling concepts.
